부위 탄성 설정

● 도형  클릭해 종류 선택 후 드래그
⬟ 자유 다각형  클릭해 점 추가, 더블클릭/Enter로 완료
🖐 이동/크기  도형을 끌어 이동, 모서리/변 핸들로 확대·늘이기
  · Shift = 비율 유지, Alt = 중심 기준
◈ 점 편집  도형 선택 후:
  · 네모/원/다이아 = 앵커 (끌어 이동)
  · 파란 원 = 접선 핸들 (곡선 조정)
  · 세그먼트 더블클릭 = 앵커 추가
  · 앵커 더블클릭/Alt+클릭 = 타입 순환
  · 우클릭/Del = 앵커 삭제
Esc: 취소 · Tab/Shift+Tab: 도형 전환
Ctrl+C/V/X/D: 복사/붙여/잘라/복제
Ctrl+Z/Y: 실행취소/다시실행 · Del: 삭제
←↑→↓: 미세 이동 (Shift = 크게)
